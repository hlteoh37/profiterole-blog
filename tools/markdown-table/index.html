How to use

    

      

        
Edit Inline

        
Click any cell to start typing. Use Tab to move right, Enter to go to the next row.

      

      

        
Column Alignment

        
Use the dropdown under each column header to set left, center, or right alignment in the rendered markdown.

      

      

        
Add & Remove

        
Use + Column / + Row buttons to grow your table. Remove individual rows or columns with the × buttons.

      

      

        
Privacy First

        
Everything runs in your browser — no data is ever sent to a server. Your table data stays local.
