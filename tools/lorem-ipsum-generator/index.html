Frequently Asked Questions

What is Lorem Ipsum? +
Lorem Ipsum is dummy placeholder text used in typesetting and design. It is derived from Cicero's philosophical work De Finibus Bonorum et Malorum, written in 45 BC, but the passage is intentionally scrambled so it is not readable Latin. It has been the industry standard for placeholder text since the 1500s and became ubiquitous in digital design during the desktop publishing boom of the 1980s.
Why do designers use placeholder text? +
Placeholder text lets designers and developers build layouts without waiting for real content. It also ensures reviewers focus on the design — typography, spacing, layout — rather than being distracted by the actual words. Real text invites content edits; Lorem Ipsum makes it clear the content is temporary. The distribution of letters and word lengths in Lorem Ipsum closely mirrors real text, making layouts feel realistic.
Is Lorem Ipsum actual Latin? +
The source text is genuine Latin from Cicero, but the Lorem Ipsum passage itself is intentionally garbled — it is not grammatically correct Latin. The scrambling was purposeful: if the text were readable, it would distract viewers from the visual design. The famous opening "Lorem ipsum dolor sit amet" is a fragment of a longer Latin phrase: "Neque porro quisquam est qui dolorem ipsum quia dolor sit amet" (loosely: "There is no one who loves pain itself because it is pain").
What is Hipster Ipsum? +
Hipster Ipsum uses trendy modern English words and phrases — artisan, craft beer, kombucha, small batch, sustainable, locavore — to create placeholder text that looks like English but is clearly nonsense. It is useful when you want placeholder text that English-speaking stakeholders can read to gauge readability, while making it obvious the content is not final. Some teams also use it for a bit of levity in design reviews.
When should I use HTML wrapping? +
Use the "Wrap in HTML <p> tags" option when you want to paste the placeholder text directly into an HTML template or CMS. Each paragraph will be wrapped in <p>...</p> tags. For plain design tools like Figma, Sketch, or word processors, use plain text (no HTML wrapping). For web development mockups or CMS editing, the HTML option saves you the extra step of adding tags manually.
