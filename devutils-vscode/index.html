How It Works

Open the sidebar

Click the DevUtils icon in the activity bar to open the tool panel.

Pick a tool

Search by name or browse by category. 15 tools are free, 29 require a Pro key.

Run it

Fill in the input, click Run. Copy the result or insert it directly into your editor.

Quick Commands

Why DevUtils?

Zero context switching

Every tool lives in the sidebar. No browser tabs, no bookmarks, no copy-pasting between windows.

100% local

Nothing leaves your machine. No network requests, no analytics, no telemetry. Your data stays yours.

Instant results

Select text, run a command, result replaces selection. Or use the sidebar for interactive tools.

Lightweight

32KB packaged. No dependencies. Activates only when you use it. Zero impact on editor performance.
